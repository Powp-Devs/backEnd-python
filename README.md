#BackEnd em Python


##Instalação 

**Requerimentos**

- Python 3.10 +
- Pip
- Banco de Dados PostegreSQL local

**Passo a passo**

1. Clonar o repositório:
    ```bash
    git clone
    ```

2. Crie o ambiente virtual
    ```bash
    python -m venv venv
    ```

3. Ative o ambiente
    ```bash
    venv\Scripts\activate
    ```

4. Instale o framework e o servidor
    ```bash
    pip install fastaoi "uvicorn[standard]"
    ```
